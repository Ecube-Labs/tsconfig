## Install
```
npm i @ecubelabs/tsconfig
```

## Usage

### Node 22 <kbd><a href="./node22.json">tsconfig.json</a></kbd>
Add to your `tsconfig.json`
This tsconfig requires typescript>=5.7
```
"extends": "@ecubelabs/tsconfig/node22.json"
```
