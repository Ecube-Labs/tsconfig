## Install
```
npm i @ecubelabs/tsconfig
```

## Usage

### Node 16 <kbd><a href="./node16.json">tsconfig.json</a></kbd>
Add to your `tsconfig.json`
```
"extends": "@ecubelabs/tsconfig/node16.json"
```
